# Vitraj Glass - Modern Website

Moderní webové stránky pro Vitraj Glass - skleněné vitráže Opočno.

## 🚀 Deploy na GitHub Pages

1. Vytvořte nový repozitář na GitHub.com
2. V terminálu v této složce spusťte:
   ```
   git remote add origin https://github.com/VASE-USERNAME/vitraj-glass.git
   git push -u origin main
   ```
3. V GitHub nastavení repozitáře > Pages > vyberte "main" branch
4. Hotovo! Web bude na: `https://vas-username.github.io/vitraj-glass/`

## 📸 Obrázky

Obrázky jsou načítány z původního serveru (vitrajglass.cz/images/) - není potřeba je stahovat.

## 🔧 Úpravy

Upravte soubor `index.html` pro změny v obsahu nebo stylu.
